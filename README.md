# Freelance Hub



## Getting started

To install my Freelance-hub Project, follow these steps:

1. Clone the repository: `git clone https://github.com/UsmonjonQahhorov/freelance-hub.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Migrate models: `1: python manage.py makemigrations, 2: python manage.py migrate`
4. Start the server: `python manage.py runserver`
